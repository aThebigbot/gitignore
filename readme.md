# Gitignore

Generate a .gitignore for your project

## Installation

```bash
git clone https:://github.com/aThebigbot/gitignore.git
cd gitignore
sh install.sh
```

## Usage

```bash
$ gitignore [template]
```

## Template registry

All templates are listed in the github official [gitignore repository](https://github.com/github/github/gitignore)
